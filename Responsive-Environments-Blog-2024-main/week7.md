# Week 7

### Labs

We began implementing the visual logic for the angel scene. Using TouchDesigner’s SOP and TOP pipelines, we simulated a soft, elastic soap-bubble effect. The blowing gesture was mapped to the bubble’s bounce-back behavior, while hand movement distorted the bubble mesh dynamically.

### Reading & Reflections

We studied the use of feedback loops and displacement effects in TouchDesigner. Applying these ideas helped us craft more responsive deformations for the bubble when interacted with. The emotional texture of visual motion became a focus this week.

---

### Peer Support

During testing, we encountered variations in gesture performance due to different user hand sizes and motion intensities. Collaboratively, we created a normalized parameter range system that adapts to user input more flexibly. Qin suggested separating facial and hand input paths for stability.

---

### Project Development

We improved the scene’s visual purity by replacing the background with a light, ethereal “heaven-like” environment. Node organization was optimized for real-time preview and quick switching between gesture responses. Work is progressing steadily on the angel mode.
